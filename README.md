# ActPilot — Path of Exile 1

Оверлей для прохождения актов PoE 1 с маршрутом, таймером, отслеживанием уровня
через `Client.txt`, деревом пассивов, камнями и ручным редактором билда.

## Быстрый запуск

Требуется Python 3.10+.

```powershell
py -3 -m pip install -r requirements.txt
py -3 app.py
```

В Windows также можно запустить `run.bat`.

## Сборка EXE

```powershell
py -3 -m pip install -r requirements.txt
py -3 -m PyInstaller --noconfirm --clean ActPilot.spec
```

Или запустить `build.bat`. Готовый файл появится в
`dist/ActPilot-PoE1.exe`.

## Структура

- `app.py` — единственная точка запуска;
- `ActPilot.spec` — сценарий сборки;
- `src/actpilot/` — пакет приложения: оверлей, дерево пассивов, камни,
  редактор билда, мониторинг `Client.txt`, стили и пути;
- `src/actpilot/assets/` — UI-ассеты и шрифты;
- `src/actpilot/data/poe1/` — дерево, описания и изображения камней;
- `tests/` — юнит-тесты чистой логики;
- `tools/` — инвентаризация эффективного поведения (`mro_inventory`);
- `requirements.txt` — зависимости запуска и сборки.

## Пользовательские данные

Настройки, прогресс и импортированные билды создаются в `src/actpilot/data/`
при запуске из исходников и рядом с EXE в собранной версии. Эти файлы
перечислены в `.gitignore`, поэтому личный путь к `Client.txt` не попадёт в
GitHub.


## Выпуск новой версии

Запустите одну команду, явно указав нужный номер версии:

```powershell
.\release.bat 1.1.0
```

Скрипт обновит `version.py`, проверит локальную сборку, закоммитит все изменения,
создаст тег `v1.1.0` и атомарно отправит коммит с тегом в `origin`. Затем GitHub
Actions автоматически соберёт Windows EXE и создаст GitHub Release.
Имя приложенного файла должно оставаться `ActPilot-PoE1.exe`: по нему встроенное
обновление находит новую сборку. Настройки, прогресс и билды в папке `data` при
обновлении не заменяются.

Собранное приложение автоматически проверяет обновления через несколько секунд
после запуска. Ошибки фоновой проверки (нет интернета или ещё нет Release) не
мешают запуску; ручная кнопка в настройках показывает подробную ошибку.
